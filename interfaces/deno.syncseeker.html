seekSync

  • seekSync(offset: number, whence: SeekMode): number
    • 定义在 lib.deno.ns.d.ts:555

    Seek sets the offset for the next readSync() or writeSync() to offset, interpreted according to whence: SEEK_START means relative to the start of the file, SEEK_CURRENT means relative to the current offset, and SEEK_END means relative to the end.

    Seeking to an offset before the start of the file is an error. Seeking to any positive offset is legal, but the behavior of subsequent I/O operations on the underlying object is implementation-dependent.

    参数

    • offset: number
    • whence: SeekMode

    返回 number
