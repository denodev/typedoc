Properties

Optional mode

mode: number
  • 定义在 lib.deno.ns.d.ts:997

创建目录时使用的权限（在调用 umask 之前，默认值为 0o777）。在 Windows 上被忽略。

Optional recursive

recursive: boolean
  • 定义在 lib.deno.ns.d.ts:995

默认为 false。 如果设置为 true，则意味着还将创建所有中间目录（如 shell 命令 mkdir -p 那样）。 使用相同的权限创建中间目录。 当设置为 true 时，如果路径中已经存在目录，或者该路径是到现有目录的符号链接，则会静默地操作成功（不更改任何权限）。
