## 实用命令

- `yarn format`: 整理文件格式。所有文件格式均使用 prettier 的默认配置。如果没有特殊必要，你基本上不需要手动运行此命令。当 `git commit` 的时候此命令会自动运行。
- `yarn build`: 生成 typedoc 文件。你可以通过在浏览器中打开 `./typedoc/index.html` 来预览最终的生成效果。
